Cita Mágica
-----------

Para el valor de la afinidad, que hemos calculado entre 0 y 100, hemos utilizado los propios valores que venían dados en el anunciado de la práctica. Algunos de estos (peso, altura) no son muy representativos, pero son suficientes.
Los valores utilizados son:
 + Diferencia de edad (10+ años, 0% - misma edad, 10% (lineal))
 + Diferencia de peso (20+ kg, 0% - mismo peso, 10% (lineal))
 + Diferencia de alturas (40+ cm, 0% - misma altura, 10% (lineal))
 + Diferencia de timidez (uno tímido y otro no, 0% - ambos tímidos o ninguno tímido, 10%)
 + Diferencia de religion (diferente religión, 0% - misma religión, 10%)
 + Redes sociales (ambos tienen perfil de las mismas redes, 10% - discrepan en una, 5% - discrepan en ambas, 0%)
 + Gusto por salir (a ambos les gusta o a ninguno le gusta, 10% - a uno le gusta y al otro no, 0%)
 + Diferenciadel número de amigos (25+ amigos, 0% - mismo amigos, 10% (lineal))
 + Carácter (mismo carácter, 10% - distinto, 0%)
 + Sociabilidad (ambos sociables o no sociables, 10% - uno sociable y el otro no, 0%)
